# Server_Client_SK
Server-client project (Computer networks labs)

### Jak uruchomić
#### Linux
 - skompilować serwer w terminalu:
   `g++ -Wall ./mancala.cpp -o ./mancalaserwer`
 - uruchomić serwer:
   `./mancalaserwer`

#### Windows(Python)
 - urochomić plik gameclient.py :
   `python gameclient.py "IP adres serwera" 8080`
